## 4. 구현내용

* `Realm`을 이용한 영상 기본 데이터 저장
* `AVPlayer`를 통한 영상 플레이어 구현
* 영상 플레이어 인터페이스 구현

## 5. 프로젝트 핵심 경험
* `SwiftUI`를 이용한 UI구현
* `AVPlayer`를 통한 영상 재생
* `AVCapture`관련 타입을 통한 영상 녹화

## 6. 아쉬운점
* Firebase를 활용한 원격 데이터베이스 구축을 하지 못한 점
* 데이터베이스의 실패에 대한 이벤트처리를 하지 못한 점
* Combine을 통한 이벤트 및 비동기처리를 하지 못한 점
